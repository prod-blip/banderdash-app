# Banderdash App

Banderdash is a local-first interactive article platform. The MVP will help a writer move from raw prose to reviewed, meaningful interactions and export a static web-component artifact.

Current status: repository scaffold for the MVP implementation.

## Development

```bash
npm install
npm run typecheck
npm test
npm run lint
```

## MVP constraints

- Single-user local/self-hosted app.
- Localhost-only editor.
- Writer approval before interactions are included.
- Library-first audited interaction components.
- Static validation is a hard gate before export.
- No hosted SaaS, LAN editor, auth, collaboration, CDN, or extra export modes in MVP.
